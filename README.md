# oboeru-blank-map

国名を覚えるための白地図です。
> https://kuansy373.github.io/anki-world-map/

### **コマンドについて**
- 右上の虫眼鏡ボタンで表示される地域名検索欄から入力できます。
- 実行ではなく入力で即時反映なのでコマンドとは言わないかもしれませんが、コマンドと言うことにしてます。
- 今後、コマンドや仕様そのものを変更するかもしれないので、「あれ、使えなくなったな」などがあればここを確認しに来てください。
> [コマンド一覧↓](#コマンド一覧)

### **地域名検索欄の説明**
- 「,」で複数検索できます。
- 「;」でコマンド入力できます。
- 「.」でコマンドにオプションを指定できます。
- コマンドは`;`で始まり、`;`または`,`で区切られます（明示的に終わらせる必要がある場合も必要です）。
  - 例1（地域名のみ） `asia,africa,oceania`
  - 例2（コマンドのみ） `;cmd1;cmd2;cmd3`
  - 例3（コマンドと地域名の混在） `asia;cmd1,africa;cmd2`
- 次の入力がコマンドの場合は`;`で、地域名の場合は`,`で区切ります。
- 地域名は大文字、小文字の区別はありませんが、コマンドは区別されます。

### **コマンド一覧**
#### **ズームボタンを表示**
1. `;zm` 画面の左右両側にズームボタン「+」「-」を表示（zoom の略）
2. `;zm.r` 右のみ表示（zoom right の略）
3. `;zm.l` 左のみ表示（zoom left の略）
4. `;zm.y数値;` ズームボタンの位置調整（明示的に終わらせる必要があるので終了の`;`または`,`が必要）
- 例: `;zm.l.y40;` ズームボタンを左のみに表示し、位置を画面下端から40%上に設定
#### **クロスヘアを表示**
1. `;aim` 画面中央にクロスヘアを表示
2. `;aim.w数値` クロスヘアの横線の長さを指定（widthの略）初期値24(px)
3. `;aim.h数値` クロスヘアの縦線の長さを指定（heightの略）初期値24(px)
4. `;aim.s数値` クロスヘアのサイズを指定（`width`と`height`を同時に指定）
5. `;aim.o数値` クロスヘアの透明度を指定（opacityの略) 初期値100(%)
- 例: `;aim.s20.o30` クロスヘアのサイズを20pxに設定し、透明度を30%に設定
#### **画面位置座標（location）を表示**
1. `;loc` 画面右下に`center: [経度, 緯度], zoom: `を表示
2. `.数値` 小数桁数を指定
